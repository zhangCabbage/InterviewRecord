## 食用说明



### 目的

面试过程是相互交流、选择的过程。

这个项目主要是为了记录各公司的面试经验，交流其中的技术问题、面试技巧等。

兵无常势，水无常形。

面经的作用绝非仅仅是投机原题，技术人最重要的是随机应变和学习的能力。

最后，祝大家都能拿到满意offer。
